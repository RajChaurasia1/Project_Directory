  • Project By: Ushmita Pareek

  • Description:
    1. The objective of the project is to select k nodes in a signed social network, taking account of both negative and positive influence of people on each other, in such a way that total influence in a network is maximum.
    2. The project is done in three parts:
      Information Propagation in the network is done using a viral market framework based on the traditional IC model and SIR model.
      The dynamic update for individuals’ beliefs, addressing the problem of parallel recommendations that involve both positive and negative relationships in particular.
      And finally, based on the PageRank algorithm and dynamic belief updates, a new Signed-PageRank algorithm is used to select the initial k nodes.
